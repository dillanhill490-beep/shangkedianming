# 课堂点名器（Web 本地版）

一个 **纯前端、零依赖、可直接本地运行** 的课堂点名器网页应用。  
适合教师、培训机构在课堂中进行快速点名与考勤记录管理。

无需后端、无需部署，双击 `index.html` 即可使用。

---

## ✨ 功能特性

### 1️⃣ 学生名单管理
- 支持导入 CSV 学生名单
- 自动去重、忽略空行、清理多余空格
- 支持有/无表头（`姓名`）的 CSV 文件
- 导入后自动保存到本地（刷新不丢失）

### 2️⃣ 点名功能
- 点名模式：
  - 🎲 随机点名
  - ➡️ 顺序点名
- 实时显示：
  - 总学生数
  - 已点名人数
  - 未点名人数
- 对当前学生可标记状态：
  - ✅ 出勤
  - ⏰ 迟到
  - ❌ 缺勤
- 标记后自动切换到下一个学生
- 所有学生点名完成后自动提示

### 3️⃣ 考勤记录管理
- 自动记录每次点名的时间戳
- 按日期范围筛选记录
- 导出考勤记录为 CSV（Excel 打开不乱码）
- 支持一键清空全部记录（二次确认）

### 4️⃣ 本地数据持久化
- 使用 `localStorage` 保存：
  - 学生名单
  - 点名记录
- 刷新页面或关闭浏览器后数据仍然存在

## 📄 CSV 文件格式说明

支持的最小格式示例：

```csv
姓名
张三
李四
王五
```

或无表头：

```csv
张三
李四
王五
```

规则说明：
- 只读取第一列
- 自动忽略空行
- 自动去重
- 自动处理 UTF-8 / BOM

---

## 📤 导出 CSV 说明

导出字段顺序为：

```
日期时间, 姓名, 状态
```

- UTF-8 with BOM 编码
- 可直接用 Excel 打开，中文不乱码
